Data Manipulation using Numpy

Manipulating Data using Numpy:
IPython | Jupyter
  • numpy.arrange()

    Function returns a ndarray containing evenly seperated values within a given range. 

                                        numpy.arange(start, stop, step, dtype)
    Arguments:
    1. start:

      Beginning of an interval

    2. stop:

      Ending of an interval.

    3. dtype:

      Datatype of the result. Default input datatype is used if none specified.

    4. step:

      Intervals between objects in array. Default spacing is 1.

  • numpy.logspace()

    This function returns an ndarray object that contains the numbers that are evenly spaced on a log scale.

    Start and stop endpoints of the scale are indices of the base, usually 10 

                                          numpy.logscale(start, stop, num, endpoint, base, dtype)
    Arguments:
    1. start

      The starting point of the sequence is basestart

    2. stop

      The final value of sequence is base stop

    3. num

      The number of values between the range. Default is 50

    4. endpoint

      If true, stop is the last value in the range

    5. base

      Base of log space, default is 10

    6. dtype

      Data type of output array if given else it'd depend on other input arguments
