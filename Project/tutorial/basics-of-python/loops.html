Loops

Loops: Repeated actions, functions and statements
IPython | Jupyter

A loop statement executes a statement or group of statements multiple times.

Types of Loops:

  1. While Loop:

    Repeats a statement or group of statements as long as a given condition is TRUE. It tests the condition before executing.

  2. For Loop:

    Executes a sequence of statements a given number of times and abbreviates the section of code that manages the loop variable.

  3. Nested Loops:

    You can nest a loop within another loop to execute different actions on a hierarchial basis.

Loop Control statements

Loop control statements change execution from its normal sequence. When execution leaves a scope, all automatic objects that were created in that scope are destroyed.

Control Statement Action
Break statement Terminates the current process.
Continue Statement Causes the loop to skip the rest of its immediate process and immediately retest its condition prior to reiterating.
Pass Statement Does literally nothng; Mostly used as a placeholder for an unwritten function or for syntactical correctness.
